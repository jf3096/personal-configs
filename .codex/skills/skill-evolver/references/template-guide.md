# Template Guide

Use this naming convention for one cycle:

- `docs/REQ-<ID>.md`
- `docs/REHEARSAL-<ID>.md`
- `docs/POSTMORTEM-<ID>.md`
- `docs/REQ-<ID>-R2.md`

Minimum content:

## REQ
- Objective
- Scope
- Acceptance criteria
- Evidence plan

## REHEARSAL
- Predicted steps
- Risk matrix
- Validation matrix (positive/negative/adversarial)

## POSTMORTEM
- What happened
- Delta vs rehearsal
- Rule/test improvements

## REQ-R2
- Tightened acceptance criteria
- Mandatory rerun checks
