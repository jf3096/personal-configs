# Workflow Graph Skill Usage

## 触发语句
- “把这个流程画成树状决策图”
- “用简版输出”
- “用详细版输出”
- “你自动判断用哪种模式”

## 模式规则
- `简版`：仅节点 + 一行解释（`::`）
- `详细版`：关键节点 + 子点（不限数量）
- 未声明模式：agent 按复杂度自动判断

## 简版示例
```text
(1) 接收任务描述 :: 汇总输入目标
└─ (2) 信息是否完整? :: 检查最小条件
   ├─ Yes -> (3) 生成树图 :: 输出图本体
   └─ No  -> (4) 追问缺失信息 :: 回补关键字段 -> [LOOP L1 -> (1)]
```

## 详细版示例
```text
(1) 接收任务描述 :: 汇总输入目标
    [1.1] 输入来源
          - 用户自然语言
          - 仓库路径或文档路径
└─ (2) 信息是否完整? :: 检查最小条件
   [2.1] 必备字段
         - 分析对象
         - 输出目标
   ├─ Yes -> (3) 生成树图 :: 输出图本体
   │          [3.1] 节点规范
   │                - `(n) 节点标题 :: 一行解释`
   └─ No  -> (4) 追问缺失信息 :: 回补关键字段 -> [LOOP L1 -> (1)]
```
